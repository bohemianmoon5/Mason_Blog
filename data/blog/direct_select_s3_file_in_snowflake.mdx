## Introduction

Snowflake에서 Copy 과정을 거치지 않고 S3 데이터를 바로 볼 수 있는 방법.

## Method

External Stage 생성 > Select문으로 데이터 조회

## Screen Shot

### 1. Select문으로 데이터 조회 (이전 포스트에서 External Stage 생성했음)

![cap_01](https://github.com/bohemianmoon5/Mason_Blog/assets/65845572/a71dfe4a-f9f3-46d3-b667-2b2edc9e0fdd)

```ex
SELECT
  $1,
  $2,
  $3,
  $4
FROM
  @<external_stage_name/file_name>;
```

## Benefits

- Copy 하기 전 간단하게 데이터를 미리 조회해 볼 수 있음
- 해당 쿼리문으로 데이터를 보며 File Format도 알맞게 수정할 수 있음
