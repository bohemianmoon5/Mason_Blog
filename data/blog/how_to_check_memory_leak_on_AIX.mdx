📌 **핵심 키워드**: `MALLOCDEBUG`

---

## 🚀 본론: AIX 환경에서 memory leak을 확인하기 위한 설정 방안

### ✅ Step 1 - 설정 방안

```ex 💻
# 환경 변수 설정

- ex) export MALLOCDEBUG=report_allocations,stack_depth:20,output:leak_mem
- report_allocations: 모든 메모리 할당 내역 추적
- output: 출력 파일 이름
- stack_depth: 호출 스택 단계 설정
```

## 📋 결론

- 간단한 환경변수 설정으로 memory leak을 관리하고 서비스를 운영하는데 도움을 줄 수 있다.
